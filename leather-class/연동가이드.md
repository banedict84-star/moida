# 📩 신청 자동화 연결 가이드 (광고 → 신청 → 자동 견적)

광고 링크로 신청을 받으면 → **구글시트에 자동 정리** + **신청자 이메일로 견적서 자동 발송** 되는 구조입니다.

```
[페북/인스타 광고] → [신청 페이지 apply.html] → 제출
   → [구글 앱스크립트] → ① 구글시트 자동정리  ② 견적서 자동메일
```

---

## 1단계. 엔진(앱스크립트) 만들기  *— 한 번만, 약 5분*

1. 구글 드라이브 → **새 구글 시트** 하나 생성
2. 시트 메뉴 **[확장 프로그램] → [Apps Script]**
3. 기본 코드 지우고 **`apps-script/Code.gs`** 내용 전체 붙여넣기
4. 맨 위 `CONFIG` 의 공방명·연락처·단가를 **실제 값으로 수정**
   - `notifyMe` 에 내 이메일을 넣으면 신청 들어올 때마다 나한테도 알림 옴
5. 오른쪽 위 **[배포] → [새 배포]**
   - 유형: **웹 앱**
   - 실행 계정: **나**
   - 액세스 권한: **모든 사용자**
   - **[배포]** → 권한 허용(내 구글계정 로그인) → **웹 앱 URL 복사**

## 2단계. 신청 페이지에 연결

1. `apply.html` 파일에서 이 줄을 찾기:
   ```js
   const ENDPOINT = "PASTE_YOUR_APPS_SCRIPT_URL_HERE";
   ```
2. 따옴표 안에 **1단계에서 복사한 URL** 붙여넣기 → 저장 → 커밋(또는 알려주시면 제가 넣어드림)

## 3단계. 테스트

- 신청 페이지 열기: `https://banedict84-star.github.io/moida/leather-class/apply.html`
- 본인 이메일로 신청해보기 → **견적 메일 도착 + 시트에 줄 추가** 되면 성공 ✅

## 4단계. 페이스북 광고에 링크 넣기

- 광고 만들 때 **'잠재고객'이 아니라 '트래픽/웹사이트'** 목표로 하고
- 도착 URL에 신청 페이지 주소를 넣기:
  ```
  https://banedict84-star.github.io/moida/leather-class/apply.html
  ```
- 인스타 프로필 링크, 블로그, 카톡에도 같은 주소를 뿌리면 됨

---

## 5단계. 신청을 관리도구 사이트에서 보기 (선택)

신청은 구글시트에 자동으로 쌓이는데, **관리도구(index.html)에서도** 자동으로 보이게 할 수 있어요.

1. 신청이 쌓이는 구글시트 → **파일 → 공유 → 웹에 게시**
2. 형식을 **'쉼표로 구분된 값(.csv)'** 으로 선택 → **게시** → 나오는 주소 복사
3. 관리도구 → **단가 설정 탭 → '신청 자동 연동(구글시트)'** 에 그 주소 붙여넣기
4. **🔄 신청 불러오기** 누르면 끝. (이후엔 도구 켤 때마다 자동으로 불러옴)

→ 이제 광고 신청이 **문의관리 목록에 자동으로 나타나고**, 거기서 견적서 출력·강사 배정·상태관리까지 됩니다.

---

## 자주 묻는 것

**Q. 견적 단가를 바꾸려면?**
→ `Code.gs` 의 `PRICES` 와 `apply.html` 의 `PRICES` **둘 다** 같은 값으로 수정 (둘이 일치해야 함).

**Q. 신청 내역은 어디서 봐?**
→ 1단계에서 만든 **구글시트**에 자동으로 쌓입니다. (관리도구로 가져오는 기능은 다음 단계에서 연결 가능)

**Q. 메일이 스팸으로 가요.**
→ 처음엔 그럴 수 있어요. 보내는 구글계정으로 평소 메일을 주고받으면 신뢰도가 올라갑니다.

**Q. 완전 무인 맞아?**
→ 네. 1·2단계 연결만 끝나면, 그 뒤로는 신청 → 정리 → 견적발송까지 사람 손 안 닿고 자동입니다.
