# 🧵 가죽공예 단체수업 자동화

학교·기관 대상 **단체 가죽공예 수업(B2B)** 운영을 자동화하는 도구입니다.
moida 프로젝트와는 별개인 독립 프로젝트예요.

## 쓰는 법
`index.html` 파일을 브라우저로 열면 끝. 설치·서버 필요 없음.
데이터는 브라우저(localStorage)에 저장됩니다.

## 자동화 범위
**재료 준비(키트)만 직접**, 나머지는 도구가 자동 처리합니다.

| 단계 | 자동화 내용 |
|------|-------------|
| 문의 접수 | 기관·인원·날짜·품목 등록 |
| 견적 산출 | 인원 × 단가 + 강사 출장비 자동 계산 (VAT 포함) |
| 정식 견적서 | 공급자 정보·견적번호·공급가/세액 들어간 한국식 견적서 인쇄/PDF |
| 강사 배정 | 인원수 기준 필요 강사 수 계산 + 지역·품목 맞는 강사 추천 |
| 서류 | 견적/계약/재방문 안내문 자동 생성 → 복사 발송 |
| 준비 체크리스트 | 단계별(재료키트 N개 등) 자동 체크리스트 |
| 일정 | 다가오는 수업 D-day 자동 정렬 |
| 재방문 | 지난 거래일 기준 대상 자동 추출 + 제안문 준비 |
| 백업 | 데이터 JSON 백업/복원 (브라우저 저장 보호) |

## 흐름
```
문의 등록 → 견적 자동생성 → 강사 자동배정 → 견적/계약 안내문 발송
   → (재료 직접 준비) → 수업 → 세금계산서/후기 → 재방문 제안
```

## 다음 단계 (완전 무인화)
지금은 "초안 자동 생성 → 복사 발송"까지. 아래를 붙이면 발송까지 무인화:
- **카카오 알림톡** — 견적/리마인드/재방문 자동 발송
- **네이버폼 연동** — 문의 자동 수집
- **Claude API** — 기관별 맞춤 안내문 자동 작성
- **전자 세금계산서** — 정산 자동 발행
